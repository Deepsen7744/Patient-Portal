# Patient Portal - Medical Document Management System

## Features
- Upload PDF files (max 10MB)
- View/download/delete documents
- Modern responsive UI

## Quick Start
1. Backend: `cd backend && npm install && npm start` (port 3001)
2. Frontend: `cd frontend && npm install && npm start` (port 3000)

## API
- POST /documents/upload
- GET /documents  
- GET /documents/:id
- DELETE /documents/:id

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: SQLite
